summary:  OFitFinder: Outdoor Gym Locator is a dedicated iOS app designed to seamlessly guide users to nearby outdoor gyms, ideal for bodyweight exercises. It simplifies the process of finding the nearest gym locations right from the app's launch, enhancing user convenience for outdoor workouts. The app boasts a straightforward interface with essential features like a menu bar, search functionality, an option to add new gym locations, and a quick navigation feature to the user's current position. Developed with Swift and SwiftUI, FitFinder ensures a smooth, efficient user experience tailored for Apple's ecosystem.
